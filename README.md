# 📄 Atividade: Artigo em LaTeX no Formato SBC

## 🎯 Objetivo

Esta atividade tem como objetivo introduzir a escrita acadêmica em LaTeX utilizando o **template oficial da SBC (Sociedade Brasileira de Computação)**.

Você deverá reproduzir um artigo a partir de um PDF apresentado em aula, utilizando corretamente:

- Estrutura de artigo científico
- Inserção de figura
- Uso de citações bibliográficas
- Organização de arquivos em um projeto LaTeX

---

## 📁 Estrutura do Projeto (OBRIGATÓRIA)

Seu repositório deve conter **exatamente os seguintes arquivos**:

```text
main.tex
referencias.bib
sbc-template.sty
sbc.bst
cameraman.png
```

### ⚠️ Observação importante

O arquivo `sbc-template.bib` é apenas um **modelo de bibliografia**.

Ele serve como exemplo de como escrever entradas bibliográficas, mas **não deve ser usado como arquivo final da atividade**.

O aluno deverá criar o seu próprio arquivo:

```text
referencias.bib
```

seguindo o padrão mostrado em `sbc-template.bib`.

---

## 📝 Tarefa

Você deverá:

1. Criar o arquivo `main.tex`
2. Reproduzir o conteúdo do arquivo artigo.pdf
3. Organizar corretamente os arquivos do projeto
4. Compilar o documento sem erros

---


## 🚫 Erros comuns que devem ser evitados

- Copiar trechos com comandos inválidos que não são LaTeX
- Escrever o nome do arquivo `.bib` errado
- Escrever `\bibliography{sbc}` em vez de `\bibliography{referencias}`
- Esquecer de incluir o arquivo `sbc.bst`
- Usar um nome diferente de `cameraman.png`
- Citar uma chave que não existe no arquivo `referencias.bib`

---

## 📦 Entrega

Você deve:

1. Subir todos os arquivos obrigatórios no repositório
2. Garantir que o projeto compile corretamente
3. Entregar o link do repositório no GitHub Classroom

---

Boa escrita e boa prática com LaTeX.
